# Game Specifications (Enemy Logic & Balance)

アクションRPGにおける、敵オブジェクトの挙動およびステータス・バランス調整用の仕様書です。
すべての数値と計算式は `script.js` 内の `Enemy` クラス初期化・更新ロジックに基づいています。

## 1. 敵ステータスのスケーリング規則
すべての敵は生成時の引数 `scaling` （現在のゲームレベルやWave数等に依存）を用いてステータスが決定されます。
以下に記載のある `(x)` は `scaling` の値を指します。

---

## 2. 敵オブジェクト一覧と詳細仕様

### 2.1. マッドゴーレム (Golem) - 近接AoE型（泥茶色の敵）
最も多く出現する標準の敵。プレイヤーに近づき、地面叩きによる円形範囲攻撃を行う。

*   **コード内タイプ名**: `golem`（デフォルト敵）
*   **基礎ステータス**:
    *   `Color`: '#5c4033' (Mud Brown)
    *   `HP`: 25 + (x * 10)
    *   `Damage`: 10 + (x * 2)
    *   `Speed`: 100 + (x * 4) [px/s]
    *   `Hitbox Radius`: 36px（表示用。元の12pxの3倍）
*   **攻撃パターン**: `circle` (地面叩き・円形範囲AoE)
    *   `攻撃射程 (Range)`: 120px
    *   `爆発範囲 (Radius)`: 140px
    *   `予兆時間 (Warning)`: 0.7秒
    *   `判定持続 (Duration)`: 0.2秒
    *   `クールダウン (Cooldown)`: 1.0秒
*   **スプライト**: `golem.png` (7コマ: Spawn / Idle / Walk / Windup / Impact / Follow-through / Death)

### 2.2. ボーンナイト (Orc) - 扇形薙ぎ払い型（骨白色の敵）
骨白の大型騎士。槍の広い扇形攻撃でプレイヤーを薙ぎ払う硬い敵。

*   **コード内タイプ名**: `orc`
*   **基礎ステータス**:
    *   `Color`: '#dcdcdc' (Bone White)
    *   `HP`: 80 + (x * 15) （ゴーレムの約3倍硬い）
    *   `Damage`: 25 + (x * 3)
    *   `Speed`: 90 + (x * 3) [px/s]
    *   `Hitbox Radius`: 54px（元の18pxの3倍）
*   **攻撃パターン**: `cone` (前方扇形・槍薙ぎ払い)
    *   `攻撃射程 (Range)`: 240px
    *   `攻撃角度 (Width)`: 126度 (Math.PI * 0.7)
    *   `予兆時間 (Warning)`: 1.0秒
    *   `判定持続 (Duration)`: 0.4秒
    *   `クールダウン (Cooldown)`: 2.0秒
*   **スプライト**: `orc.png` (7コマ: Spawn / Idle / Walk / Windup / Impact / Follow-through / Death)

### 2.3. ファントム (Mage) - ワープ近接型（紫の幽霊）
足のない浮遊する死霊。画面のかなり遠くからプレイヤーを捕捉し、ワープして広範囲爆発を起こすトリッキーな敵。
ボロ布のようなローブを纏い、紫や青白い燐光を放つ。

*   **コード内タイプ名**: `mage`
*   **基礎ステータス**:
    *   `Color`: '#3c2b5c' (Dark Purple)
    *   `HP`: 15 + (x * 8) （非常に脆い）
    *   `Damage`: 25 + (x * 5) （一撃が非常に重い）
    *   `Speed`: 40 + (x * 2) [px/s] （徒歩移動は極めて遅い）
    *   `Hitbox Radius`: 36px（元の12pxの3倍）
*   **攻撃パターン**: `teleport_burst` (ワープ＆爆発)
    *   `索敵/ワープ射程`: 600px
    *   `爆発範囲 (Radius)`: 150px
    *   `ワープ後・予兆時間`: 1.0秒
    *   `判定持続 (Duration)`: 0.5秒
    *   `クールダウン (Cooldown)`: 4.0秒
*   **ランダムワープ移動**: 1〜3秒に1回、短距離ワープで位置を変える
*   **スプライト**: `mage.png` (5コマ: Spawn / Idle / Walk / Attack / Death)

### 2.4. ボス (Boss) - フェーズ移行型
定期的に出現する大型ボス。フェーズによって攻撃パターン（扇形、弾幕、召喚など）が動的に変化する。

*   **コード内タイプ名**: `boss`
*   **基礎ステータス**:
    *   `HP`: 800 + (x * 200) （膨大）
    *   `Damage`: 40 + (x * 10) （即死級ダメージ）
    *   `Speed`: 90 + (x * 5) [px/s]
    *   `Hitbox Radius`: 75px（元の25pxの3倍）
*   **基本攻撃パターン**: `cone` (前方扇形・近接薙ぎ払い)
    *   ※フェーズ進行やタイマーにより、`summon`（手下召喚）や `spread`（弾幕散布）へ移行。
    *   `攻撃射程 (Range)`: 240px
    *   `攻撃角度 (Width)`: 90度 (Math.PI / 2)
    *   `予兆時間 (Warning)`: 1.0秒
    *   `判定持続 (Duration)`: 0.4秒
    *   `クールダウン (Cooldown)`: 1.8秒
*   **スプライト**: `boss.png` (5コマ: Spawn / Idle / Walk / Attack / Death) ※今後更新予定

## 3. 敵のスポーン確率

| タイプ | 出現確率（スケーリング依存） |
|--------|------|
| マッドゴーレム (golem) | デフォルト（他の敵が選ばれなかった場合） |
| ボーンナイト (orc) | 15% + (x * 1.5%) |
| ファントム (mage) | 5% + (x * 1%)、上限20% |
| ボス (boss) | サバイバルタイマー終了時に出現 |

## 4. その他の共通仕様
- 敵は画面端から一定の `margin`（X:60px, Y:60px等）を空けた内部にスポーンする。
- プレイヤーとの距離が近すぎる場所（Dist < 180）には出現しない。
- 被弾時（のけぞりヒットストップ）は被弾方向に応じた `knockback` が適用される。
- ボスが召喚する手下はレベル1の `golem`（マッドゴーレム）。
