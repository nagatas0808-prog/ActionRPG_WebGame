# Phase 4 Implementation Plan: Roadmap Focus

Phase 3の振り返りとロードマップに基づき、以下のコンテンツ拡張を正確に実装します。

## Proposed Changes

### 1. パークの統合と新機能
`script.js` 内のパークリストを再編し、以下の「10種類の強力な『統合パーク』」として実装します。

- **【剣聖】**: リーチ ＋ 範囲拡大
- **【神速】**: 移動速度 ＋ 回避距離
- **【絶影】**: クリティカル率 ＋ クリティカルダメージ
- **【強欲】**: 獲得経験値 1.5倍
- **【明鏡止水】**: ジャスト回避でHP回復 ＋ スロー延長
- **【金剛】**: のけぞり無効 ＋ 低HP時に攻撃力大幅アップ
- **【乾坤一擲】**: カウンター（回避直後の攻撃）強化 ＋ ダメージ反射
- **【奥義の真髄】**: SP上昇量アップ ＋ 奥義範囲拡大
- **【緊急回避】**: 被弾直前にSPを消費して自動回避
- **【連撃】**: 攻撃速度アップ
